Smart Code Explanation Demo

See how our AI explains code in simple, beginner-friendly language

Input Code:

function fibonacci(n) { if (n <= 1) return n; return fibonacci(n - 1) + fibonacci(n - 2); } console.log(fibonacci(10));

AI Explanation:

🤖 Smart Explanation:

What this code does: This function calculates Fibonacci numbers using recursion.

How it works:

  • Base case: If n is 0 or 1, return n directly (Fibonacci of 0 = 0, Fibonacci of 1 = 1)
  • Recursive case: For any other number, return the sum of the two previous Fibonacci numbers
  • Example: fibonacci(5) = fibonacci(4) + fibonacci(3)

Result: console.log(fibonacci(10)) will output 55, which is the 10th Fibonacci number.

Note: This recursive approach is simple but inefficient for large numbers. In practice, you'd use iteration or memoization for better performance.
